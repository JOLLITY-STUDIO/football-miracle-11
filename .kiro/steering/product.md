# Product Overview

Football Miracle 11 (神奇十一人 / Magic Number Eleven) is a lightweight strategy football card battle game based on the board game "Magic Number Eleven". Players deploy athlete cards, use tactical icons and synergy cards to engage in offensive and defensive battles on an 8x4 field map, with the higher score winning.

## Core Concept

The game's core mechanic revolves around shooting with a total power of 11 being an absolute victory - hence "Magic Number Eleven". The game combines elements of 21 (blackjack) where reaching 11 points guarantees success.

## Key Features

- Card-based football strategy game with tactical depth
- 8x4 field grid with zone-based positioning
- Tactical icon system (Attack, Defense, Pass, Press)
- Synergy card system for power enhancement
- Control marker system (0-100%) affecting available actions
- Draft phase for star player selection
- Substitution mechanics and injury time
- Both PvE (AI) and PvP modes
- Progressive Web App (PWA) with offline support

## Target Audience

Players who enjoy strategic card games with football themes. The game balances accessibility for casual players with depth for hardcore strategy enthusiasts.

## Design Philosophy

Follow the principles from "爆款方法论：游戏设计的三大核心与150个技巧" by 曾宜凯:
- Activate adrenaline during battle (pressure and challenge)
- Provide endorphin rewards after success (achievement feedback)
- Stimulate dopamine through immediate positive reinforcement
- Ensure clear visual feedback and intuitive mechanics
- Prioritize entertainment value-to-cost ratio
