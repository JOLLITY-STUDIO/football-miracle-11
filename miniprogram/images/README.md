# 图片资源说明

本目录用于存放微信小游戏的图片资源。

## 目录结构

```
images/
├── logo.png              # 游戏Logo
├── share.png            # 分享图片
├── cards/               # 卡牌图片
│   ├── players/         # 球员卡
│   ├── synergy/         # 协同卡
│   └── penalty/         # 点球卡
├── icons/               # 图标
├── backgrounds/         # 背景图
└── ui/                  # UI元素
```

## 图片要求

- 格式：PNG、JPG、WEBP
- 大小：建议单个图片不超过 500KB
- 尺寸：根据实际需求调整

## 卡牌尺寸建议

- 球员卡：164x360px
- 协同卡：120x180px
- 点球卡：120x180px

## 注意事项

1. 图片资源会占用小程序包大小，建议使用 CDN
2. 可以使用 webp 格式减小文件大小
3. 建议使用图片压缩工具优化图片质量
