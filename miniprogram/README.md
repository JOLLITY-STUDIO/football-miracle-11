# 足球奇迹11 - 微信小游戏

## 项目简介

足球奇迹11是一款策略卡牌对战游戏，现已支持微信小游戏平台。

## 项目结构

```
miniprogram/
├── game.js                 # 游戏入口文件
├── game.json               # 游戏配置文件
├── project.config.json    # 项目配置
├── utils/                 # 工具类
│   ├── cards.js          # 卡牌数据
│   ├── gameConfig.js     # 游戏配置
│   ├── gameLogic.js      # 游戏核心逻辑
│   └── gameUtils.js      # 游戏工具函数
├── images/               # 图片资源
│   ├── cards/           # 卡牌图片
│   ├── icons/           # 图标
│   └── ui/              # UI元素
└── README.md             # 项目说明
```

## 开发说明

### 前置要求

1. 微信开发者工具
2. Node.js 环境

### 开发步骤

1. **安装依赖**
   ```bash
   npm install
   ```

2. **配置小游戏**
   - 打开微信开发者工具
   - 选择"小游戏"项目类型
   - 导入项目，选择 `miniprogram` 目录
   - 在 `project.config.json` 中配置你的 `appid`

3. **调试游戏**
   - 点击"编译"按钮
   - 在模拟器中查看效果
   - 使用调试工具查看日志

### 游戏流程

1. **菜单** - 开始游戏
2. **选秀** - 选择3轮明星卡
3. **阵容选择** - 选择10名首发和3名替补
4. **比赛** - 进行卡牌对战
5. **结果** - 查看比赛结果

## 游戏规则

### 基本规则
- 每方10名首发球员，3名替补
- 4个区域（Zone 0-3）
- 上半场10回合，下半场10回合
- 通过传球、逼抢、射门等战术进行对战

### 控球态
- 进攻态（0-2）：最多使用3张协同卡
- 常态（3-7）：最多使用2张协同卡
- 防守态（8-10）：不能使用协同卡

### 球员类型
- 前锋（FW）：高攻击力，可放置在Zone 1-2
- 中场（MF）：平衡型，可放置在Zone 2-3
- 后卫（DF）：高防守力，可放置在Zone 3-4

## 技术架构

### 游戏引擎
- 基于 Canvas 2D API
- 自定义游戏循环
- 场景管理系统

### 场景系统
- MenuScene - 菜单场景
- DraftScene - 选秀场景
- SquadScene - 阵容选择场景
- GameScene - 游戏主场景
- ResultScene - 结果场景

### 渲染系统
- 2D Canvas 渲染
- 自定义UI组件
- 响应式布局

### 输入系统
- 触摸事件处理
- 按钮点击检测
- 手势识别

## 注意事项

1. 微信小游戏包大小限制为 4MB
2. 图片资源建议使用 CDN 或分包加载
3. 音频文件需要使用微信音频 API
4. 需要在微信公众平台配置服务器域名
